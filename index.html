The Fox Browser

The fox browser is good because

  • it can access the internet
  • it has a fox
  • not sketchy

People like this browser mainly because it's completely open source and it doesn't have tracking etc. It uses a bit more RAM than most browsers, but it has it's own task manager to help you see what uses it. Volunteers update it and they're doing a great job. Yay!
